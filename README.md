# Interactive 3D Particle System

A real-time interactive 3D particle system controlled by hand gestures.

## Features
- **Hand Tracking**: Control particle expansion/contraction by opening/closing your hand.
- **3D Shapes**: Switch between Heart, Flower, Saturn, Buddha, and Fireworks.
- **Visuals**: Glowing particles with bloom effect.
- **Customization**: Change particle colors in real-time.

## How to Run

1. Install dependencies (if not already done):
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser at the URL shown (usually `http://localhost:5173`).
4. Allow camera access when prompted.
5. Show your hand to the camera to interact!

## Controls
- **Shape Buttons**: Click to morph particles into different shapes.
- **Color Picker**: Change the color of the particles.
- **Fullscreen**: Toggle fullscreen mode.
